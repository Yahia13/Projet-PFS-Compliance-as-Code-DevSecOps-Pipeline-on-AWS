# Projet-PFS-Compliance-as-Code-DevSecOps-Pipeline-on-AWS
Créer une architecture AWS automatisée avec Terraform + Ansible, où une pipeline Jenkins scanne : Scanne Terraform (AWS misconfig) Scanne les images Docker (vulnérabilités) Scanne Kubernetes manifests (mauvaises pratiques) Déploie sur EKS uniquement si tout est conforme (Compliance as Code)
